# 🩺 SuperDoctor: The Future of Healthcare
**[Live Site →](react-project-3roo.vercel.app)**
SuperDoctor is a premium, high-performance healthcare portal designed to revolutionize the patient-doctor experience. With a futuristic dark-mode aesthetic inspired by modern high-energy design, it provides an immersive interface for managing health, booking consultations, and accessing AI-driven triage.

---

## 🚀 Features

### 🧠 AI-Driven Symptom Triage
A smart assessment tool that helps users understand their symptoms and provides immediate guidance on the next steps, whether it's self-care or urgent consultation.

### 🏠 Home Visit Integration
Seamless map-based scheduling for home visits. Patients can request a doctor to visit their location with real-time tracking and location services.

### ⚡ Emergency Mode
A rapid-response module designed for critical situations, providing instant access to emergency services and immediate medical protocols.

### 📅 Smart Calendar Booking
An intuitive, calendar-based interface for scheduling appointments with specialists, managing follow-ups, and viewing availability at a glance.

### 🌿 Wellness Hub
A dedicated space for health education, featuring:
- **Interactive Training**: High-performance YouTube previews with hover-activated previews.
- **Health Articles**: Categorized, expert-reviewed articles for proactive health management.

### 🏎️ High-Energy Landing Page
A visually stunning landing experience featuring:
- **Sticky Minimizing Hero**: Dynamic scaling on scroll for a premium feel.
- **Horizontal Scroll Transitions**: "About" sections that transition seamlessly from vertical to horizontal movement.
- **Neon Aesthetic**: High-contrast, futuristic design language.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/) (Vite)
- **Styling**: [TailwindCSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router Dom 7](https://reactrouter.com/)
- **Data Visualization**: [Recharts](https://recharts.org/)

## ⚙️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
